### 关于歌曲URL   
按照``song.json`` 中给的示例编写歌曲的URL信息   
```JSON
[
    {
        "title": "audio name",
        "url": "audio url"
    },
]
```

>>目前仓库里的json文件中的url指向对应的存储桶我已经删了  
>>json文件命名默认为``song.json``，想改别的文件名可以去``randomplay.js``中将 song.json 替换成要更改的文件名
